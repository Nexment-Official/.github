# Pull Request

Thank you for taking the time to contribute to Nexment! ❤️

Before continuing, please note that **Nexment is not currently accepting code contributions as part of its normal development process.**

The projects are still evolving rapidly, and the maintainers are responsible for keeping the architecture and codebase consistent.

If your pull request contains code, it may be reviewed. If the proposed changes align with the project's direction, maintainers may adapt the implementation to match the current architecture and coding style before merging. All accepted contributions will receive full credit.

For most improvements, we recommend opening an **Issue** instead of a pull request.
